## Содержимое портфолио
### 1) Test Documentation
Папка: `Test Documentation`
- `Check_list.xlsx` — чек-лист для web-приложения
- `Test_cases.pdf` — набор тест-кейсов
- `bug_reports.pdf` — оформленные баг-репорты (STR, AR/ER, severity/priority)
### 2) API Testing
Папка: `API Testing`
- `Postman_Collection.json` — коллекция запросов
- Проверки: статус-коды, schema/JSON validation, негативные сценарии
- Методы: GET/POST/PUT/DELETE
### 3) SQL / DB
Папка: `DB requests`
- `Работа с SELECT в MySQL.xlsx` — примеры SELECT-запросов ( фильтрация)
