# 🎯 Портфолио QA Engineer (Junior)

Привет! Меня зовут Василий. В этом репозитории собраны примеры моих практических работ по ручному тестированию web-приложений и API. 

Все артефакты были созданы в рамках интенсивного обучения и максимально приближены к реальным коммерческим задачам.

## 📂 Что внутри

### 1. Тестовая документация (`/Test_Documentation`)
Примеры создания понятной и структурированной документации:
* **Test_cases.pdf** — проектирование тест-кейсов для web-приложения. Применял техники тест-дизайна: классы эквивалентности и анализ граничных значений.
* **Bug_reports.pdf** — примеры оформления баг-репортов. Содержат четкие шаги для воспроизведения (Steps to reproduce), ожидаемый/фактический результат и окружение.

### 2. Тестирование API (`/API_Postman`)
Практика работы с REST API:
* **API_Collection.json** — экспортированная коллекция из Postman. 
  * Реализованы базовые методы: `GET`, `POST`, `PUT`, `DELETE`.
  * Написаны скрипты-проверки (Tests) на статус-коды ответа (200, 201, 400, 404) и валидацию структуры JSON.
  * *Для просмотра: импортируйте этот файл в свой Postman.*

### 3. Основы Python (`/Python_Scripts`)
* **test_script.py** — пример написания базовых скриптов на Python для понимания логики автоматизации и работы с кодом.

## 🛠 Технологии и навыки
* **Инструменты:** Postman, Chrome DevTools
* **Теория:** Клиент-серверная архитектура, модель OSI, жизненный цикл разработки ПО (SDLC)
* **Техники тест-дизайна:** Эквивалентное разделение, Граничные значения, Попарное тестирование
* **Языки и VCS:** Python, Git
