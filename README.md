# QA Portfolio - Василий Луцек

Привет! Это мое практическое QA-портфолио.
Здесь собраны материалы, с которыми я обычно работаю как junior QA: тест-документация, API-тестирование в Postman и SQL-практика.

Проект сделан как "рабочая папка" с реальными артефактами, а не только как красивый шаблон.

## Что внутри

- `Test Documentation/Check_list.xlsx` - чек-лист для web-приложения.
- `Test Documentation/Test_cases.pdf` - набор тест-кейсов.
- `Test Documentation/bug_reports.pdf` - примеры оформленных баг-репортов.
- `API Testing/REST_Collection.json` - Postman-коллекция для тестирования REST API.
- `API Testing/README_DemoShopping.md` - инструкция по запуску коллекции и переменным.
- `API Testing/SOAP_Collection.json` - Postman-коллекция для тестирования SOAP API.
- `DB requests/Работа с SELECT в MySQL.xlsx` - SQL-практика по базовым SELECT-запросам.

## Что я здесь отрабатываю

- написание и запуск API-тестов (позитивных и негативных);
- работу с токенами, переменными и окружениями в Postman;
- проверку контрактов и структуры ответов API;
- базовые проверки данных через SQL;
- аккуратное оформление QA-артефактов под портфолио и отклики.

## Быстрый старт

1. Импортируйте `REST_Collection.json` в Postman.
2. Настройте переменные окружения .
3. Запустите `login` или `register`, чтобы получить токен.
4. Прогоните сценарии из `Products` и `Cart`.
5. Для деталей используйте `README_DemoShopping.md`.

## Коротко про стек

- **Инструменты:** Postman, Swagger/OpenAPI, SQL, Chrome DevTools
- **База:** клиент-серверная архитектура, HTTP, тест-дизайн
- **Дополнительно:** Git/GitHub, базовый HTML/CSS
